# タスク管理アプリ

## 目的
社内の3名チームが日々のタスクを管理するためのWebアプリ。
付箋のようにタスクカードを「未着手」「進行中」「完了」の3列にドラッグ＆ドロップで移動できるカンバンボード。

## ユーザー
- 社内チーム3名（小貫、田中、鈴木）
- 管理者は小貫のみ

## 機能一覧

### 認証
- ユーザー名とパスワードでログイン
- ログアウト
- 初期ユーザーは上記3名をシードデータとして作成

### カンバンボード
- 3列構成: 未着手 / 進行中 / 完了
- タスクカードをドラッグ＆ドロップで列間移動
- タスクカードには以下を表示: タイトル、担当者、期限、優先度（高/中/低を色で表現）

### タスク管理
- タスクの新規作成（タイトル、説明、担当者、期限、優先度）
- タスクの編集
- タスクの削除（確認ダイアログ付き）
- 担当者でフィルタリング

### ダッシュボード
- 全タスク数、未着手/進行中/完了の件数
- 担当者別のタスク件数
- 期限切れタスクの一覧（赤で強調）

## その他の要件
- レスポンシブ対応（PCとスマホ）
- ダークモード切替
- データはSQLiteに保存
