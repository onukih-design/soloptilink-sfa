### 提案4: TypeScript型安全性の回復（優先度: P1）

#### 現状
`any`型が4ファイルに残存:
- `src/hooks/use-companies.ts`
- `src/hooks/use-deals.ts`
- `src/hooks/use-orders.ts`
- `src/lib/utils/excel-export.ts`

#### 放置した場合
- **1ヶ月後**: 型エラーが伝播して他のファイルにもany汚染
- **3ヶ月後**: TypeScriptの恩恵が半減

#### 推奨解決策
1. `use-companies.ts`: `let companies: any[]` → `Company[]`に変更
2. `use-deals.ts`: sort関数に型ガード追加
3. `use-orders.ts`: `let orders: any[]` → `AIToolOrder[] | SalesOutsourcingOrder[]`に変更
4. `excel-export.ts`: `data: any[]` → ジェネリクス`<T>(data: T[])`に変更

#### 期待効果
型安全性100%回復

#### 所要工数
1人で0.5日
