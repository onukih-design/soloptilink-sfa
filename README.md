## v5.2: グローバルナレッジ共有

```
プロジェクトA: 初回 → 知見蓄積 → グローバル(~/.soloptilink/)に同期
                                      ↓
プロジェクトB: セットアップ時に自動インポート → 初回から過去の全知見で武装
                                      ↓

```

もう「よくあるミス」を繰り返さない。プロジェクトをまたいで知恵が引き継がれます。

## ナレッジエンジン

```
1回目のプロジェクト: ナレッジベース = 初期知見のみ
  → レビューで問題発見 → ナレッジに自動蓄積

2回目のプロジェクト: ナレッジベース = 初期 + 1回目の学習
  → 1回目で見つけた問題を重点チェック → 新しい知見も蓄積

5回目のプロジェクト: ナレッジベース = 累計5プロジェクト分の知見
  → もう「よくあるミス」は開発セッションの時点で防げる
  → レビューはより高度な品質改善に集中

→ これが「疑似カスタムLLM」。SoloptiLink専用の開発知見で武装したClaude。
```

## クイックスタート

```bash
unzip Claude_自律開発キット_v5_2.zip -d my-project
cd my-project && chmod +x setup.sh && ./setup.sh

# ↑ 過去のグローバルナレッジがあれば自動インポートを提案

# チェーン開発（推奨）
./chain.sh "タスク管理ツール作って"

# ナレッジ統計
./chain.sh --stats           # ローカル
./chain.sh --global-stats    # 全プロジェクト横断
```

## チェーンの流れ

```
Session 1 [DEVELOP] ← ナレッジ注入: BP + AP
  PM: 知見に基づいた高品質開発
  ↓

Session 2 [REVIEW] ← ナレッジ注入: 全知見
  レビュアー: 過去の問題パターンを重点チェック
  ↓ スコア: 72/100

Session 2.5 [LEARN] ← 自動実行
  🧠 レビュー結果から新しい知見を抽出 → learned.md に蓄積
  ↓

Session 3 [FIX] ← ナレッジ注入: BP
  修正: ベストプラクティスに沿った修正
  ↓

Session 4 [RE-REVIEW] → スコア: 91/100 → 完了！
  🧠 さらに知見蓄積
```

## ナレッジベース構成

```
~/.soloptilink/knowledge/           ← ★グローバル（全プロジェクト共有）
  ├── global-learned.md              ← 全プロジェクトの知見を統合
  ├── global-calibration.md          ← 全プロジェクトのスコア履歴
  └── projects.log                   ← 登録プロジェクト一覧

my-project/docs/knowledge/          ← ローカル（プロジェクト固有）
  ├── review-patterns.md             ← 頻出問題パターン（P001〜）
  ├── best-practices.md              ← 開発ベストプラクティス（BP001〜）
  ├── anti-patterns.md               ← やってはいけないこと（AP001〜）
  ├── scoring-calibration.md         ← スコア基準 + 過去スコア履歴
  └── learned.md                     ← 自動蓄積知見（★使うたびに成長）
```

### ナレッジの流れ
```
[セットアップ] グローバル → ローカル（過去の全知見をインポート）
[チェーン中]   ローカルで知見を蓄積
[チェーン完了] ローカル → グローバル（新知見を自動同期）
                ↓
[次のプロジェクト] グローバルの蓄積がさらに増えた状態で開始
```

## コマンド一覧

| コマンド | 動作 |
|---------|------|
| `./chain.sh "指示"` | 全自動（開発→2ラウンドレビュー修正） |
| `./chain.sh "指示" --rounds 3` | 3ラウンド |
| `./chain.sh --review-only` | レビューだけ |
| `./chain.sh --fix-only` | 修正だけ |
| `./chain.sh --continue` | 前回の続き |
| `./chain.sh --stats` | ローカルナレッジ統計 |
| **ナレッジ共有** | |
| `./chain.sh --sync` | ローカル→グローバルに手動同期 |
| `./chain.sh --import` | グローバル→ローカルにインポート |
| `./chain.sh --global-stats` | 全プロジェクト横断統計 |
| `./chain.sh --compact-global` | グローバルナレッジをLLMで圧縮 |
